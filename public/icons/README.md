# PWA Icons for HomFud

This directory contains the Progressive Web App (PWA) icons for HomFud.

## Required Icons

Please create PNG icons in the following sizes and place them in this directory:

- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels

## Design Guidelines

- Use the HomFud brand colors (orange theme: #f97316)
- Include a food-related icon or the HomFud logo
- Ensure good contrast and readability at small sizes
- Use a simple, clean design that works well as a maskable icon

## Tools for Icon Generation

You can use online tools like:
- https://favicon.io/favicon-generator/
- https://www.favicon-generator.org/
- https://favicon.io/favicon-converter/

Or design tools like:
- Figma
- Adobe Illustrator
- Canva

## Icon Requirements

- Format: PNG
- Background: Transparent or solid color
- Color scheme: Match the app's orange theme (#f97316)
- Style: Modern, clean, food-related
